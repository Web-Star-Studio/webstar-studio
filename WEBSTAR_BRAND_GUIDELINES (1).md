## Color Palette

### Primary Colors

| Color Name | Hex Code | RGB | Usage |
|------------|----------|-----|-------|
| **Neon Lime** | `#cfff28` | rgb(207, 255, 40) | Primary accent, CTAs, highlights |
| **Deep Space** | `#060022` | rgb(6, 0, 34) | Primary background, dark mode |

### Secondary Colors

| Color Name | Hex Code | RGB | Usage |
|------------|----------|-----|-------|
| **Soft White** | `#f3f8f0` | rgb(243, 248, 240) | Light backgrounds, text on dark |
| **Lavender Mist** | `#d2c5ff` | rgb(210, 197, 255) | Secondary accent, subtle highlights |
| **Deep Teal** | `#345c59` | rgb(52, 92, 89) | Tertiary accent, supporting elements |

### Color Application Rules
- **Dark backgrounds** (`#060022`) serve as the primary canvas
- **Neon Lime** (`#cfff28`) for emphasis, calls-to-action, and brand moments
- **Soft White** (`#f3f8f0`) for body text on dark backgrounds
- **Lavender Mist** (`#d2c5ff`) for secondary information and subtle accents
- **Deep Teal** (`#345c59`) for supporting graphical elements

## Typography

### Primary Typeface: Raleway
**Usage:** Headlines, display text, brand moments  
**Characteristics:** Modern, geometric sans-serif with elegant thin weights  
**Sample:** "The quick brown fox jumps over the lazy dog"

### Secondary Typeface: Codec Pro
**Usage:** Body copy, UI elements, supporting text  
**Characteristics:** Clean, professional, highly readable  
**Sample:** "The quick brown fox jumps over the lazy dog"

### Typography Hierarchy
| Element | Typeface | Weight | Notes |
|---------|----------|--------|-------|
| H1 Headlines | Raleway | Light/Regular | Large display sizes |
| H2-H3 Subheads | Raleway | Medium | Section headers |
| Body Text | Codec Pro | Regular | Paragraph content |
| UI/Labels | Codec Pro | Medium | Interface elements |
| Accents | Raleway | Bold | Emphasis points |

## Visual Elements

### Brand Symbols
The brand utilizes abstract, geometric symbols featuring:
- **Line-art constructions** with the neon lime color
- **Spherical/orbital shapes** suggesting motion and technology
- **Dot matrices** and grid patterns
- **Wireframe 3D forms** representing digital space
- **Star/spark motifs** connecting to the "star" in Webstar

### Imagery Style
**Theme:** Space exploration, astronauts, cosmic environments  
**Treatment:** High contrast, desaturated backgrounds with neon accents  
**Subjects:** Astronauts, planets, nebulae, rockets, metallic textures

### Symbol Applications
- Use as decorative elements in layouts
- Apply as section dividers or accent graphics
- Incorporate in presentations and digital materials
- Maintain neon lime (`#cfff28`) as primary symbol color

### Content Generation Guidelines
When creating content for Webstar:
1. Lead with vision and possibility
2. Use active, confident language
3. Emphasize transformation and results
4. Maintain technical credibility without jargon
5. Balance authority with approachability
6. Reference space/exploration metaphors where appropriate
7. Avoid generic agency/service language
